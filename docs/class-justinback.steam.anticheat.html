public boolean
# RequestPlayerGameBan( string $sCheatDescription, integer $iDuration, boolean $bDelayBan = false )

Requests a game ban on a specific player.

Requests a game ban on a specific player.

This is designed to be used after the incidents from ReportPlayerCheating have been reviewed and cheating has been confirmed.

Parameters

$sCheatDescription
Text describing cheating infraction.
$iDuration
Ban duration requested in seconds. (duration 0 will issue infinite - less than a year is a suspension and not visible on profile)
$bDelayBan
(Optional) Delay the ban according to default ban delay rules.

Returns

boolean
TRUE if the player has successfully been banned otherwise FALSE

Throws

justinback\steam\exceptions\SteamRequestException
if the servers are down, or the web request failed
justinback\steam\exceptions\SteamRequestParameterException
if the steam id, report id, or duration is not valid as a parameter
justinback\steam\exceptions\SteamException
if the app id or api key is not valid as a parameter
